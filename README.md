# Voyage Travel

Bienvenidos a **Voyage Travel**, una empresa especializada en Turismo comprometida en brindar servicios turísticos con seriedad, confiabilidad, amabilidad y distinción. Este proyecto ha sido desarrollado como un esfuerzo grupal utilizando tecnologías como HTML, CSS, JavaScript, y librerías como Bootstrap y Bootstrap Icons.

## Descripción del Proyecto

Voyage es una Empresa de Viajes y Turismo que se destaca en el mercado por su amplio compromiso hacia los clientes. Nuestra principal misión es ofrecer un servicio personalizado de calidad y confiabilidad, a través de la buena atención de nuestro personal debidamente capacitado, diseñando viajes únicos, a precios accesibles y superando las expectativas de nuestros clientes.

El sitio web incluye una sección de administración donde un administrador puede ingresar con un usuario y contraseña para agregar, editar y eliminar paquetes de vuelos.

## Despliegue

## Características

- **Administración de Paquetes de Vuelos**: Un administrador puede agregar, editar y eliminar paquetes de vuelos mediante una interfaz sencilla y amigable.
- **Diseño Responsivo**: La página está diseñada para ser completamente responsiva y accesible desde cualquier dispositivo.
- **Interfaz de Usuario Atractiva**: Utilizamos Bootstrap y Bootstrap Icons para crear una interfaz moderna y atractiva.

## Instalación y Uso

1. **Clonar el Repositorio**:
    ```bash
    git clone https://github.com/turepositorio/voyage-travel.git
    ```
2. **Navegar al Directorio del Proyecto**:
    ```bash
    cd voyage-travel
    ```
3. **Abrir el Archivo `index.html` en un Navegador**.
